# Layout Clone
- 사이트의 메인 레이아웃을 클로닝하며 SCSS 응용 능력 향상

<br/>

## 👩‍💻 구현 내용 
- SCSS로 스타일을 작성
- 작성된 SCSS를 CSS로 컴파일
- 레거시 코드 활용보단 최신의 CSS Flex와 Grid를 활용
- 시멘틱 태그를 최대한 활용
- BEM 방법론 활용
